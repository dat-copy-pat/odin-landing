# odin-landing
The Odin Project Foundations Project: Landing page

The project is an assigment for the Foundations Course from The Odin Project.
https://www.theodinproject.com/lessons/foundations-landing-page

# Assignment #
To create an entire web page from a design provided. 
- image of complete website
- brief of fonts and colors used

# Steps #
1. Create the structure
2. Add the content, placeholders and resets
3. Style header section
4. Style information section
5. Style quote section
6. Style footer

## Note ##
The assignment does not require making this project responsive.
